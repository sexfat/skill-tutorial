# 檔案摘要（省 token 示範版）

1. 用 Read 工具讀取使用者指定的那個檔案（僅限該檔案，不要額外 `ls`／`find`／讀取其他檔案，除非摘要必須依賴的 import 對象使用者也要求了解）。
2. 用 5 條以內的條列式摘要回覆：這個檔案的用途、關鍵函式/區塊、值得注意的地方（若有）。
3. 不要覆述檔案全文，不要主動提供重構/測試/安全建議，除非使用者另外要求。

完成即結束，不需要額外確認或提出後續選項。
